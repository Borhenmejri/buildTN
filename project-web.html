About This Project
Built as a Final Year Project for Esprit School of Business, this Django web application serves as the interface layer for a complete industrial data analytics system at CASCO Automotive Tunisia (Amphenol Group).
Multi-dataset CRUD with pagination for 140,000+ records across SCRAP, maintenance, and HSE modules
Embedded Power BI dashboards with iframe integration for live reporting
Prophet 5-year forecasting, Random Forest risk classification, and Isolation Forest anomaly detection
SQL Server backend via mssql-django with optimized queries for large datasets
Bootstrap 5 responsive UI with Font Awesome icons and custom dark theme
Full Scrum methodology with 4 sprints — ETL, EDA, ML, and final deployment
Tech Stack
Django + Python
Backend framework — views, models, URL routing, template engine
SQL Server + mssql-django
Database — casco_db on DESKTOP-ART0JK0, optimized for large tables
Bootstrap 5 + Font Awesome
Responsive UI components, icons, and custom dark theme styling
Power BI (iframe embed)
4 dashboards embedded — SCRAP KPIs, maintenance, predictive analytics
Scikit-learn + Prophet
ML pipeline — forecasting, risk classification, anomaly detection
